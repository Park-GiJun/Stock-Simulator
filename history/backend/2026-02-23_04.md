# 2026-02-23_04: LLM 의존성 제거 및 스케줄러 비활성화

## 변경 사항

### 1. 뉴스 생성 LLM 제거 → 랜덤 템플릿 방식으로 변경
- `NewsContentGeneratorWithLLM`에서 Anthropic Claude LLM 호출 완전 제거
- 섹터별(IT, 농경, 제조업, 서비스업, 부동산, 럭셔리, 식료) 사전 정의된 뉴스 템플릿 리스트 추가
- 이벤트 레벨별(COMPANY, INDUSTRY, SOCIETY) 템플릿 분류
- 각 템플릿에 sentiment, intensity 범위, duration 범위 포함
- 랜덤 선택 후 범위 내 랜덤 값으로 뉴스 생성

### 2. 매매 결정 LLM 제거 → 확률 계산 방식으로 변경
- `TradingDecisionGeneratorWithLLM`에서 Anthropic Claude LLM 호출 완전 제거
- 투자 성향(AGGRESSIVE/STABLE/VALUE/RANDOM)별 매수/매도 기본 확률 설정
- 리스크 허용도에 따른 확률 보정
- 최근 뉴스 감정(POSITIVE/NEGATIVE)에 따른 확률 보정
- 보유 종목 수에 따른 확률 보정
- 매수 시: 선호 섹터 우선, 긍정 뉴스 섹터 가중치 적용
- 매도 시: 투자 성향별 매도 비율 차등 적용

### 3. 스케줄러 비활성화
- `StockListingScheduler`: IPO/상장폐지 스케줄러 `@Scheduled` 주석 처리
- `NpcCreationScheduler`: 개인투자자 생성 스케줄러 `@Scheduled` 주석 처리
- `InstitutionCreationScheduler`: 기관투자자 생성 스케줄러 `@Scheduled` 주석 처리
- 뉴스 생성 및 NPC/기관 트레이딩 스케줄러는 유지
