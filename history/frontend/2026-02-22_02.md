# 2026-02-22_02: NPC/기관투자자 포트폴리오 상세 페이지 추가

## 변경 사항

### 1. 투자자 상세 페이지 (데스크탑 + 모바일)
- `/investors/[type]/[id]` 라우트 추가 (type: `npc` 또는 `institution`)
- 투자자 프로필 카드: 이름, 유형, 투자성향, 거래빈도, 자본금, 수입, 위험허용도, 선호섹터
- 자산 현황: 총 자산, 보유 현금, 주식 평가금, 총 손익
- 보유 종목 테이블: 종목명(링크), 보유수량, 매입가, 현재가, 손익, 비중
- 최근 거래 내역: 매수/매도 필터, 종목명, 날짜, 수량, 가격
- 모바일 동일 기능: `/m/investors/[type]/[id]` 라우트

### 2. 투자자 목록 페이지 클릭 연동
- 데스크탑: 기관/NPC 테이블 행 클릭 시 상세 페이지로 이동
- 모바일: 카드 헤더 클릭 시 상세 페이지로 이동

### 3. portfolioHelper 수정
- `getEnrichedPortfolio()` 에 `investorType` 파라미터 추가 (기본값 'USER')
- NPC/INSTITUTION 타입으로 trading-service API 호출 가능

## 수정 파일 목록
